# VESA Converter for ED-HMI3010-070C

A 3D-printable VESA mount adapter for the ED-HMI3010-070C display.

## What is this?

An STL file that converts your ED-HMI3010-070C screen to work with standard VESA mounting arms and stands.

## How to use

1. Download the STL file
2. Print it on your 3D printer
3. Attach it to your display
4. Mount on any standard VESA arm or stand

## Print settings

- Layer height: 0.2mm
- Infill: 15-20%
- Material: PLA or PETG

## Files

- `VESAConverterforED_HMI3010-070C-Cut.stl` - The adapter model
